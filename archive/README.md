# Electronic Notarization Facility (ENF) Documentation

This repository contains comprehensive documentation for building and operating an **Electronic Notarization Facility (ENF)** in the Philippines, covering architecture, compliance, and development planning.

## Documentation Structure

- **ENF Strategy** - Accreditation requirements, technical stack, and implementation plan
- **Customer Journey** - User experience flows for notarization services
- **Notarization Flow** - Dual-path canonicalization and notarization processes
- **Technical Architecture** - System design, security, and compliance frameworks

## View Documentation

To run the documentation locally:

```bash
pip install -r requirements.txt
mkdocs serve
```

Then visit http://localhost:8000

## Building

To build the static site:

```bash
mkdocs build
```

## About ENF

The Electronic Notarization Facility platform provides a modern, secure, and extensible framework for digital notarization in the Philippines under the Rules on Electronic Notarization (A.M. No. 24-10-14-SC).

**Key Features:**
- HyperVerge eKYC integration
- TRUTH Engine verification
- Blockchain anchoring (optional)
- Compliance with Philippine regulations
- Support for Remote and In-Person Electronic Notarization
